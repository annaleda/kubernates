
# CKAD Pod Design

## Teoria

* **Jobs**: task batch una tantum.
* **CronJobs**: task schedulati.
* **Rolling Updates**: aggiornamenti progressivi.
* **Rollback**: ripristino versione precedente.

## Esercizi

1. Creare Job che stampa "Hello CKAD".
2. Creare CronJob ogni minuto.
3. Aggiornare immagine deployment.
4. Eseguire rollback.
